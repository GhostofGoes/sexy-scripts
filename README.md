# Introduction
A collection of scripts I use for setting up my environment and doing day-to-day tasks.

# Terminal Environment Setup

Linux
```
cd ./sneaky-scripts
./setup.sh
```

Windows
```
cd ./sneaky-scripts
./setup.ps1
```
